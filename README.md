# Sleeping Bag (Fabric 1.21.11)

Adds a **Sleeping Bag** item crafted from a **Bundle + any Bed** (shapeless).
Right click with the sleeping bag in your hand to skip to morning **without changing your respawn point**.

## Build (Windows)
- `.\gradlew.bat build`

## Build (PowerShell also works)
- `./gradlew build`

Requires Java 21+ to compile (Java 22 is fine if you compile with `--release 21`).
